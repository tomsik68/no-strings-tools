About

The "no strings attached" suite of apps exists to solve a number of everyday tasks. It makes zero sense to install dedicated (desktop or mobile) apps if we just want to check time across timezones, perform everyday calculations or even transfer a small snippet of text between devices. Your browser can do these tasks just fine. No trackers, no

What these apps promise

  • No accounts.: no signup, no login
  • No tracking.: no ads that slow down your browser
  • Offline-first.: once everything is cached, these apps generate no extra network traffic
  • One problem per tool.: tools remain small and uncluttered

Who makes this

I'm Tomáš Jašek. I make these apps for my own needs and share with everyone. The code is open, ready to be extended by anyone. Every app is a folder with an HTML file and a bit of vanilla JavaScript that one person can read in a sitting.

If these tools save you time, you can sponsor the project. No pressure - everything here stays free either way.
